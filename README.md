# QueueLess

A Smart Queue Management System that helps users book and manage queues online.

## Features

- User Login & Registration
- Hospital Queue
- Bank Queue
- Salon Queue
- Canteen Queue
- Admin Dashboard

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## How to Run

1. Install dependencies:
   npm install

2. Start the server:
   node server.js

3. Open the frontend in your browser.
